## Major ongoing projects
* I help write documentation and plan ideas for [@SideStore](https://github.com/SideStore) ([website](https://sidestore.io)), an alternative iOS/iPadOS app store for seamless sideloading. No jailbreak needed, and no computer after the initial install, only a Wi-Fi connection.
* I help develop [Juxtaposition](https://github.com/PretendoNetwork/juxtaposition), [Pretendo Network](https://pretendo.network)'s [Miiverse](https://miiwiki.org/wiki/Miiverse) replacement and successor. ([Pretendo Network](https://pretendo.network) is a free and open source replacement for Nintendo's servers for both the 3DS and Wii U.)
